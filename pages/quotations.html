Quotações e citações

Nesse capítulo vamos dar uma olhada nos elementos <blockquote> <q> <abbr> <adress> <cite> e <bdo>.

Exemplo

Aqui está uma citação do site da W3C:

HTML e CSS são as tecnologias fundamentais para o desenvolvimento de páginas Web: HTML (html e xhtml) para a estrutura, CSS para o estilo e leiaute, incluindo WebFonts.
Código

<p>Aqui está uma citação do site da W3C:</p>

<blockquote cite="https://www.w3c.br/Padroes/WebDesignAplicacoes"> HTML e CSS são as tecnologias fundamentais para o desenvolvimento de páginas Web: HTML (html e xhtml) para a estrutura, CSS para o estilo e leiaute, incluindo WebFonts. </blockquote>
